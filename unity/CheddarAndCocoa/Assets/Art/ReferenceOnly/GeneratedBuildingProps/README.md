# Generated Building Prop Pack

Deterministic transparent cartoon house and yard-building sprites generated by `tools/art/generate_building_prop_pack.py`. Runtime code layers these over the existing house/patio and back-door Unity objects so the playable yard reads as a home exterior instead of stacked geometric district markers. These are generated couch-test assets, not a final hand-authored building art pipeline.
