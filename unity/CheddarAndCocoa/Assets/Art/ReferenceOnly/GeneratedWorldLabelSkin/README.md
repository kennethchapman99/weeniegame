# Generated World Label Skin Pack

Deterministic transparent cartoon skins generated by `tools/art/generate_world_label_skin_pack.py`. Runtime TextMesh strings, positions, and mission-controller state remain authoritative, but mission labels and score pops draw these sprites behind the text instead of reading as raw floating placeholder text.
