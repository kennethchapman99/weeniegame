# Generated HUD Skin Pack

Deterministic transparent cartoon HUD skin sprites generated by `tools/art/generate_hud_skin_pack.py`. Runtime IMGUI text and button hitboxes remain authoritative, but major mission-select, briefing, end-card, session-summary, and playtest overlay surfaces use these sprites instead of flat white-rectangle boxes. These are couch-test UI assets, not a final hand-authored menu system.
