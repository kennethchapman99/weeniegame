# Meta Game Loop

The game should reward replay without becoming a progression treadmill.

## Core Loop

Play mission.
Earn stars.
Unlock cosmetic rewards.
Unlock new missions.
Improve ranks.
Replay for mastery.

## Unlock Types

Cosmetics only.

Examples:
- collars
- bandanas
- bow ties
- sweaters
- silly hats
- toy skins
- dog bed themes
- victory poses

No gameplay advantage.

## Collectibles

Collectible sets create long-term goals.

Examples:
- squirrel evidence
- lost toys
- famous sticks
- mysterious socks
- neighborhood smells

## Profile Stats

Track lifetime stats.

Examples:
- squirrels chased
- toys rescued
- united barks
- zoomies triggered
- pool falls
- fake snacks eaten

## Session Design

Ideal session:
- 10 to 30 minutes.

A player should always have:
- one mission to beat;
- one challenge objective to chase;
- one collectible set to progress.

## Design Rule

Replayability should come from:
- mastery;
- variety;
- humor.

Not from repetitive grinding.
