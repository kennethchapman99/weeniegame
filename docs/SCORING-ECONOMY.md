# Scoring Economy

> **Status: DEFERRED DESIGN REFERENCE.** Maintain existing behavior as needed, but do not expand the
> scoring economy before the deep-slice couch-playtest gate passes.

Goal: score should teach players how to play well.

## Principles

Good score events reward:
- teamwork
- timing
- objectives
- rescue behavior
- funny moments

Bad score events punish:
- spam
- mistakes
- ignored objectives

## Score Weighting

Small Event
25-100 points

Medium Event
100-300 points

Major Event
300-1000 points

## Teamwork Bonus

Whenever possible, teamwork should be worth more than solo success.

Examples:
- united bark
- partner rescue
- synchronized objective completion

## Anti-Spam Rule

Repeated identical actions should lose value.

Examples:
- barking repeatedly
- standing still in score zones
- farming easy actions

## End Of Mission

Mission score should be determined by:
- objective completion
- teamwork bonuses
- challenge objectives
- mistake penalties

Not by time spent.

## Readability Rule

Every score change must have:
- visible number;
- readable label;
- clear cause.
