# Mechanic Modules

Goal: avoid building fifteen bespoke levels.

Every mission should be assembled from reusable modules.

## Module: Herding

Used by:
- Great Backyard Squirrel Conspiracy
- Squirrel Dash

Core concepts:
- route nodes
- pressure
- cutoff zones
- fake-outs
- escape counters

Metrics:
- successful herds
- successful cutoffs
- fake-outs triggered
- escapes prevented
